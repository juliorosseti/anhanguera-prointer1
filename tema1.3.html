Sistemas Desktop on-line

Como o próprio nome já sugere, são sistemas que podem ser operados em qualquer navegador web, podendo dar ao usuário liberdade de criar conteúdo novo, pasta e quaisquer conteúdos, muitas vezes usando suportes de serviços como Dropbox por exemplo. Abaixo segue alguns destes serviços:


S.O. Funcionalidades Suporte Novos Apps (S/N)
ZeroPC Serviço Desktop on-line S
Eye OS Acesso Desktop on-line de qualquer dispositivo (PC, Tablet, Smartphone S
Glide OS Integração com vários decides e serviços Cloud; Gerenciamento de arquivos. S

Funcionalidades do Sistema Desktop on-line

Pode-se dizer que uma das maiores vantagens deste sistema é poder ter acesso aos seus arquivos em qualquer lugar com internet, também destaca-se a colaboração em tempo real entre usuários, Sistemas Desktop on-line cada vez caminham para trabalhar como um sistema desktop tradicional, aplicativos como o AbiWord, o Zoho, o Google Docs, o Etherpad e até o Microsoft Office já permitem colaboração online para tarefas comuns de escritório.

Desktop Tradicional vs Desktop on-line


Desktop on-line
Vantagem Desvantagem
- Portabilidade (executa em variados tipos de sistemas operacionais, tais como Linux e Windows) e custos com licenças.
- Desenvolvimento, manutenção e atualização centralizada da aplicação. Você não precisa sair instalando sua aplicação em diversos equipamentos diferentes. Basta colocá-lo no servidor para que os usuários a acessem; 		- A interface HTML pode se comportar diferente dependendo do navegador. Além de não ser rica em controle gráficos e peca em vários aspectos (posicionamento, performance, etc).
- Falhas na conexão prejudicam o serviço.

Desktop Tradicional
Vantagem Desvantagem
- Não depende da conexão com a internet.
- Total controle sobre o posicionamento dos controles na aplicação
- O desempenho para uma interface gráfica é mais rápido em uma aplicação desktop que usa o processamento local.
- Integração para com vários hardwares é muito mais fácil. 		- A integração com usuários remotos é pior em relação ao Desktop on-line.
- Necessário instalar sua aplicação em diferentes tipos de máquinas, com diferentes tipos de sistemas, drivers e periféricos.
- A manutenção e atualização de sua aplicação requerem um esforço extra.
